## üöÄ Getting Started

### 1. Update Your Extension
Smart Scout is built into the latest version of the Chrome Extension.
1.  Open Chrome and click the **Extensions (puzzle piece)** icon.
2.  Select **Manage Extensions**.
3.  Ensure **Developer mode** is ON (top right).
4.  Click **Update** to get the latest version.

### 2. Check the New Tab
Open the extension popup by clicking the Jiggly icon. You'll now see two tabs at the top:
*   **Posting**: Your familiar cross-listing tools.
*   **Smart Scout**: The new dashboard for monitoring deals.

## üí° Key Features & How to Use Them

### Feature 1: Real-Time Price Intelligence
**What it does:** Instantly tells you if an item is a good deal compared to eBay sold listings.

**How to use it:**
1.  Browse **Facebook Marketplace** or **Craigslist** as usual.
2.  Click on any item listing to view details.
3.  Look for the **Smart Scout Overlay** in the bottom-right corner.
4.  It shows:
    *   **eBay Average Price**: What similar items sell for.
    *   **Deal Meter**: Visual indicator (Green = Great Deal, Red = Overpriced).
    *   **Potential Profit**: The estimated spread between the asking price and market value.

### Feature 2: Watchlists (Passive Monitoring)
**What it does:** Automatically checks marketplaces for items you want and notifies you when they appear.

**How to use it:**
1.  Go to your **Dashboard** and click **Watchlist** in the navigation menu.
2.  Click **+ Add Watchlist**.
3.  Enter your keywords (e.g., "Herman Miller Aeron", "iPhone 15 Pro").
4.  Set your filters:
    *   **Price Range**: Min/Max price.
    *   **Platforms**: Facebook, Craigslist, or both.
5.  Click **Create**.
6.  *Tip: Keep the extension active or a marketplace tab open for background monitoring to work best.*

### Feature 3: Quick Reply Chips
**What it does:** Helps you message sellers faster with pre-written, polite inquiries.

**How to use it:**
1.  On Facebook Marketplace, click **Message** on a listing.
2.  Look for the blue **Quick Reply** buttons above the text box.
3.  Click a chip like **"Is it available?"** or **"Pick up today?"** to instantly fill the message box.
4.  Hit send!

## üõ†Ô∏è Troubleshooting

**"I don't see the price overlay."**
*   Ensure you are on a specific item page, not the search results.
*   Refresh the page.
*   Check if you are logged into the extension.

**"Watchlist notifications aren't appearing."**
*   Make sure browser notifications are enabled for Chrome.
*   The extension checks periodically (every 15-30 mins). Ensure Chrome is open.

**"I see a database error."**
*   If you see an error about "tables not found," please contact support. The system administrator needs to run a one-time database update.
